Using this in Blender

Go to the Shading workspace, switch the shader type to World, add an Environment Texture node, and load the file — projection is already Equirectangular. Connect it to the Background node's Color input.

The .hdr file carries real dynamic range if you shot with bracketing on; the .png is safest for reflections/backdrop when you didn't.
